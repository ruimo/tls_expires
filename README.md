# Show TLS expiration

Show expiration date of TLS certificate.

## Prerequisite

Install openssl.

## Usage

    USAGE:
        tls_expires <host_name>
    
    FLAGS:
        -h, --help       Prints help information
        -V, --version    Prints version information
    
    ARGS:
        <host_name>    Host name to check the TLS expiration. Ex: www.ruimo.com
